## Printing Instructions

### For best results (priority order):

1. **Paper:** Thick matte textured sticker paper (300gsm+). Avoid glossy vinyl.
2. **Open** `print_labels.html` in Google Chrome
3. **Print:** `Ctrl+P` → Set margins to **None** → Enable **Background Graphics** → Paper: A4
4. **Cut** along the hairline corner marks on each label

### Label Dimensions
- Each label: ~80mm × 110mm (fits 10ml attar dram bottle)
- 2 labels per row, 3 rows per A4 page = 6 labels per sheet

## Design Language

- **Typography:** Cormorant Garamond (serif) for fragrance names · Inter (sans) for brand/details
- **Heritage Elements:** Fine-line SVG Mughal arches · monochromatic botanical sketches
- **Foil Simulation:** Antique brass / tarnished copper / faded emerald tones
- **Rule:** "Quiet Luxury" — negative space is the design
